# Repuck Hockey — Presentation

## Share your exact slides (same as you see locally)

1. Open the deck in your browser (Live Server or the link below).
2. Click **Share my version** (top right).
3. Save the downloaded `presentation-edits.json` into this folder (replace the old file).
4. In Terminal:

```bash
cd "/Users/willnorton/econ presneation "
bash publish.sh
```

5. Send this link — others see **your** slides, not the default:

**https://wnorton26-sketch.github.io/repuck-hockey-presentation/**

> The live link only updates after step 4. Edits stored only in your browser are not on the link until you run **Share my version** + **publish.sh**.

## Edit locally

- **Edit slides** or open with `?edit=1`
- **Save** / **Link save file** for work on your computer
